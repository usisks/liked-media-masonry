# サポート

## 対応できる問い合わせ

- GitHub Releaseからの導入、更新、削除
- Xの「いいね」ページでの表示・抽出不具合
- ボード、ライトボックス、追加読込、設定に関する不具合
- 機能要望と文書の改善提案

## Issue作成前の確認

- 最新のGitHub Releaseを使用している
- Chromeと対象ページを再読み込みした
- Xへログインし、自分の「いいね」ページを開いている
- 同じ内容のIssueがない
- [既知の制限](docs/KNOWN_LIMITATIONS.md) に該当しない

XのDOM変更により、以前動作していた抽出が突然動かなくなる場合があります。

## 不具合報告に必要な情報

- Chromeのバージョン
- 拡張機能のバージョン
- 発生したページの種類
- 再現手順
- 期待した動作と実際の動作
- コンソールエラーまたは「エラー報告をコピー」の内容

アカウント名、投稿本文、非公開URL、Cookie、セッショントークン、メールアドレスなどの個人情報・認証情報を貼らないでください。脆弱性は公開Issueではなく [SECURITY.md](SECURITY.md) に従って報告してください。
