検証日：2026年7月13日

## 工程の目的

v0.14.3の実行時挙動を変更せず、実X確認前に大量表示、段階追加、DOM交換、動画要素交換、反復操作を合成DOMで検証できるストレス監査を追加する。

## 変更範囲

- `manifest.json`と`package.json`のバージョンを0.14.4へ更新
- `tests/run_stress_tests.py`を追加
- `npm run test:stress`と`npm run test:all`を追加
- `docs/STRESS_TEST_REPORT.md`を追加
- README、公開チェックリスト、実X手動テスト表を更新
- 通常回帰テストでストレススクリプト、文書、package scriptsを静的検査

## 実行時コードの同一性

- `content.css`
- `content/*.js`

- `popup.js`
- `popup.css`
- `icons/*.png`
- `PRIVACY.md`

DOM抽出、ボード、ライトボックス、動画、追加読込、設定、診断処理の挙動変更はない。
