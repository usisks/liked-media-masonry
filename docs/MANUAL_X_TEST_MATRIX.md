# 実X手動回帰テスト表

対象バージョン：0.15.0

テスト実施時はChrome版、Firefox版、Floorp版をそれぞれ専用プロファイルで使用します。結果欄にはブラウザ名と `PASS`、`FAIL`、`BLOCKED` のいずれかを記録してください。Firefox・Floorpの旧版からの自動更新は日常利用プロファイルを使わず、別途専用プロファイルで確認します。

> `docs/STRESS_TEST_REPORT.md`の1,175件・5,000件検査は合成DOM上の結果です。X-34とX-35は、実際のXと通常Chromeプロファイルで別途実施してください。

| ID | 項目 | 手順 | 期待結果 | 結果・証跡 |
|---|---|---|---|---|
| X-01 | 起動 | 自分のいいね欄を開き、右下ボタンまたはポップアップからボードを開く | ボードが開き、X本体との二重スクロールバーが出ない |  |
| X-02 | 終了導線 | ボード表示中に左上の「Xの表示に戻る」を押す | ボードが閉じ、Xの通常表示へ戻る |  |
| X-03 | 通常画像 | 画像投稿を含む範囲を表示して再走査 | 画像カードが重複せず表示される |  |
| X-04 | 複数画像 | 1投稿に複数画像がある投稿を表示 | 各画像が別カードとして1回ずつ表示される |  |
| X-05 | 縦長画像 | 縦長画像カードを確認 | 上下が切れず、元の縦横比を維持する |  |
| X-06 | ホバー情報 | カードへマウスを重ねる | 投稿者名・ユーザーID・投稿日だけが表示され、本文は常時表示されない |  |
| X-07 | 画像ライトボックス | 画像カードを開く | 同一画面で拡大され、中央画像から元投稿を開ける |  |
| X-08 | 前後移動 | 左右ボタン・左右キーで移動 | 画像・動画・GIFを順番どおり移動し、末尾から先頭へループしない |  |
| X-09 | 左右プレビュー | 前後のプレビュー画像をクリック | 画像部分だけがクリック対象となり、指定メディアへ移動する |  |
| X-10 | 閉じる操作 | Esc、閉じるボタン、暗幕クリックを試す | いずれもライトボックスを閉じる |  |
| X-11 | フォーカス | キーボードでカードを開き閉じる | 開始時は閉じるボタン、終了時は元カードへフォーカスが戻る |  |
| X-12 | 背景ホイール | ライトボックス中にホイール操作 | 前後移動ではなく背景ボードだけがスクロールする |  |
| X-13 | 背景キー操作 | ↑↓、Page Up/Down、Spaceを使用 | 背景ボードが滑らかに移動し、ブラウザ標準のCtrl＋ホイールは妨げない |  |
| X-14 | 閉じた後の位置・維持 | 「スクロール先をそのまま表示」を選び、背景を移動して閉じる | 移動先のボード位置を維持する |  |
| X-15 | 閉じた後の位置・復元 | 「開く前の位置に戻る」を選び、背景を移動して閉じる | 開いた時点のボード位置へ戻る |  |
| X-16 | 復元中断 | 位置復元直後にユーザーが再スクロール | 強制復元が中止され、ユーザー操作を優先する |  |
| X-17 | 動画カード | 動画を含む範囲で動画表示をオンにする | 高さが潰れず、サムネイルまたは代替表示が出る |  |
| X-18 | MP4動画再生 | 通常動画をライトボックスで開く | 可能な場合は専用プレイヤーで再生する |  |
| X-19 | GIF再生 | GIFをライトボックスで開く | 前後移動に含まれ、ミュート・ループで再生する |  |
| X-20 | 動画切替 | 動画→画像→別動画と素早く移動 | 離れた動画が停止し、古いイベントが現在動画へ影響しない |  |
| X-21 | 動画フォールバック | 再生元を取得できない動画を開く | 黒い0:00プレイヤーを残さず、サムネイルと「Xで再生する」を表示 |  |
| X-22 | 通常の追加読込 | ボード最下部まで手動スクロール | 追加読込が1回だけ開始し、件数または追加なしをトースト表示 |  |
| X-23 | 2回目の追加読込 | 追加後に再び最下部へ移動 | 2回目以降も新しい投稿を取得できる |  |
| X-24 | 読込中の再要求 | 読込中にポップアップ・ボタン・下端操作を重ねる | 並列実行せず、次回要求を最大1件だけ予約する |  |
| X-25 | ライトボックス中の読込 | 拡大表示を開いたまま背景を最下部へ進める | 表示中メディアを維持したまま追加読込する |  |
| X-26 | 読込中断 | 読込中にボードを閉じる | X側を動かす処理を停止し、開いた時点のX位置へ戻る |  |
| X-27 | 手動ボタン | ポップアップとライトボックスの「続きを読み込む」を使用 | 同じLoadController経路で実行し、状態表示が一致する |  |
| X-28 | 追加なし | 新規メディアがない状態で複数回試す | 追加なしを表示し、自動読込が一時休止する |  |
| X-29 | 引用投稿 | 引用投稿を含むいいねを表示 | 外側投稿のstatus URLを優先し、引用元と混同しない |  |
| X-30 | SPA移動 | いいね欄→ホーム→別プロフィール→いいね欄と移動 | Observer・イベント・タイマーを解除・再接続し、重複UIを作らない |  |
| X-31 | X通常画面の保全 | ボードと動画を使用後にX表示へ戻る | Xのスクロール、動画、投稿DOMが壊れていない |  |
| X-32 | 設定保持 | 設定を変更してタブ・Chromeを再読み込み | 4設定が保持される |  |
| X-33 | 診断情報 | エラー報告をコピーしてJSONを確認 | URL、投稿本文、投稿者名、完全UAが含まれない |  |
| X-34 | 500件以上 | 500件以上を読み込んで操作 | 著しい操作不能、無限タイマー、動画要素増殖がない |  |
| X-35 | 長時間 | 30分以上、読込とライトボックス操作を繰り返す | CPU・メモリが継続的に増え続けない |  |
