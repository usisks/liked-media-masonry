# 既知の制限

- XのDOM、属性名、URL構造、動画配信方式が変更されると、抽出・追加読込・再生が動作しなくなる可能性があります。
- ログイン状態、A/Bテスト、地域、年齢、センシティブメディア設定により表示構造が異なる場合があります。
- 通常URLの動画を取得できず、X側の `blob:` 動画要素も破棄済みの場合、拡張機能内では再生できません。サムネイルと元投稿リンクを表示します。
- `poster` だけが存在する動画は、再生可能な動画URLを取得できない場合があります。
- Xの非公開APIへ直接アクセスしないため、画面へ読み込まれていない投稿やメディアは取得できません。
- 画面外カードの画像・動画は解放しますが、カード外枠はDOM上に残ります。大量表示では件数に応じて負荷が増えます。
- 実際のX上での長時間回帰テストと全表示形式の確認は自動化していません。
- GitHub配布版はChromeウェブストア審査済みではありません。

本拡張機能はXまたはPinterestの公式製品ではなく、各社から承認、提供、推奨されたものではありません。
