# 権限の説明

## `storage`

カード幅、動画・GIF表示、ライトボックスを閉じた後の位置、左右プレビュー透明度を `chrome.storage.local` に保存するために使用します。

投稿、メディアURL、いいね履歴、アカウント情報は保存しません。`localStorage` は使用しません。

## `x.com` と `twitter.com` へのサイトアクセス

`manifest.json` のcontent script対象を次のURLへ限定しています。

- `https://x.com/*`
- `https://twitter.com/*`

Xの「いいね」ページで、利用者のブラウザが既に表示または読み込んだDOMから画像、動画サムネイル、投稿情報を読み取るために必要です。対象外サイトではcontent scriptを実行しません。

## 使用しない権限・機能

- Cookie、履歴、タブ全体、ダウンロード、クリップボードの常時権限
- background service worker
- Xの非公開API
- 外部JavaScript
- 広告、解析、追跡

診断情報のコピーはポップアップ操作に応じたブラウザ機能として実行し、自動送信しません。

## 外部通信

開発者または第三者のサーバーへデータを送る処理は実装していません。Xが既に読み込んだメディアURLをブラウザ内表示に利用する場合がありますが、新しい解析APIや変換APIへ問い合わせません。
